🩺 Diabetes Health Data Analysis Project
📌 Overview
This project analyzes a medical dataset to understand the relationship between health indicators and diabetes. The goal is to identify key factors that contribute to diabetes risk using data analysis techniques.

 📊 Dataset
The dataset used is the **Pima Indians Diabetes Dataset**, which includes medical measurements such as:
- Glucose level
- Blood pressure
- BMI
- Age
- Insulin levels
- Diabetes outcome (0 = No diabetes, 1 = Diabetes)

 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Google Colab
📌 Steps Performed

1. Data Loading
The dataset was loaded directly from an online source into a Pandas DataFrame.
2. Data Cleaning
- Invalid zero values in medical fields were replaced with NaN
- Missing values were filled using median values

3. Exploratory Data Analysis
- Statistical summary of the dataset
- Correlation analysis between variables

4. Group Analysis
- Age groups vs diabetes outcome (crosstab analysis)
- BMI categories vs diabetes outcome
 📈 Key Findings
- Glucose level and BMI are strongly associated with diabetes risk
- Older individuals have a higher probability of diabetes
- Obese individuals show significantly higher diabetes cases
- Lifestyle and metabolic factors play a major role in diabetes prediction
 🎯 Conclusion
This analysis demonstrates how data science can be used in healthcare to identify patterns and risk factors for diseases. The findings can help support early detection and better understanding of diabetes risk factors.
📁 How to Run
1. Open the notebook in Google Colab
2. Run all cells from top to bottom
3. Ensure no errors occur
